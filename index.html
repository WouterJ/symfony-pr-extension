Symfony PR Extension - A small extension that adds PR templates when creating a PR

Installation

  1. Open chrome using the --enable-easy-off-store-extension-install flag
  2. Download and install the extension
  3. Go back to contributing!

Usage

This extension detects when you are submitting a pull request for a repository that is using the Symfony rules[1]. It'll add the code or documentation pull request template in the description.

Conventions

It'll also try to fill in some answers, based on some conventions.

Bug Fixes

When a branch is called fix_12, fix-12 or fix/12, the extension assumes it is fixing a bug reported in issue #12.

Adding Features

When a branch starts with add (and doc or document for documentation repos), the extension assumes it is adding a new feature.
